# SerialController

Copyright (C) 2021 Xie Yang. All rights reserved. 

### 介绍
​	串口遥控器（适配VS Code）

### 使用说明
1. 先输入串口号
2. w,s,a,d 控制速度与转向
3. shift 电机断电
4. space 刹车

### 编译环境
- Windows 10
- Visual Studio Code
- gcc version 8.1.0 (x86_64-posix-seh-rev0, Built by MinGW-W64 project)
